## 2026-04-04 - Path Traversal Vulnerability in UploadFile\n**Vulnerability:** The `UploadFile.filename` property in FastAPI is directly controlled by the client and can contain directory traversal characters like `../../../`. Using it directly with `pathlib.Path` to construct a file path allows attackers to escape intended directories and write arbitrary files to the system.\n**Learning:** Validating the parent directory is not enough if the filename itself contains traversal strings that evaluate outside the directory. Always treat client-provided filenames as untrusted.\n**Prevention:** Extract only the basename of the uploaded file using `Path(file.filename).name` before concatenating it with the target directory path.
